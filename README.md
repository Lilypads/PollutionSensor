## How to use the program

1. Clone the repository

`git clone https://github.com/Lilypads/PollutionSensor`

2. Get inside the directory

`cd /path_to_directory/PollutionSensor`

3. Compile here

`cmake .`

`make`

4. Run the program

`./fileName`
